# Calculadora-de-preços-de-imóveis

# Objetivo
- Criar um site que calcule os preços de imóveis com exatidão e eficiência implementando algoritmos de aprendizado de máquina nele.

  # Tecnologias a serem utilizadas
  - Python
  - Javascript
  - HTML
  - Flask
  - Requests
  - python-dotenv
  - Pandas
  - Numpy
  - Render_template
  - Seaborn
  - Matplotlib
  - SQL
  - PostgreSQL
