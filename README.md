# Calculadora de preços de imóveis

# Objetivo
- Criar um site 100% desenvolvido em Programação orientada a objetos em Python que calcule os preços de imóveis com exatidão, implementando algoritmos de aprendizado de máquina nele.

  # Tecnologias a serem utilizadas
  - Python
  - Pandas
  - Numpy
  - Tkinter
