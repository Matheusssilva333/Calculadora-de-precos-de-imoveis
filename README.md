# Calculadora de Empréstimo Imobiliário

# Objetivo
- Criar um site 100% desenvolvido em Programação orientada a objetos em Python que faça cálculos de simulação de empréstimos imobiliários, implementando algoritmos de aprendizado de máquina nele.

  # Tecnologias a serem utilizadas
  - Python
  - Pandas
  - Numpy
  - Tkinter
