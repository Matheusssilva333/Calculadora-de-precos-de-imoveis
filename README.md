# Calculadora de Empréstimo Imobiliário

# Objetivo
- Criar um site 100% desenvolvido em Programação orientada a objetos em Python que faça cálculos de simulação de empréstimos imobiliários, integrado com um agente de inteligência artificial com função de consultor de investimentos.

  # Tecnologias a serem utilizadas
  - Python
  - Pandas
  - Numpy
  - Tkinter
  - Agno
